Legacy schema definition for claims migration.
This repository simulates a real-world data migration from a legacy P&C claims system to Guidewire ClaimCenter.
- 01_legacy_schema – Legacy database schema
- 02_sample_data – Sample legacy claims data
- 03_mapping_document – Legacy to Guidewire mappings
- 04_sql_transformation – SQL extraction and cleansing
- 05_gosu_examples – Gosu migration logic
- 06_validation_reconciliation – Data validation and reconciliation
